## Currency Converter App

- Currency Converter App is a simple and intuitive application that allows users to convert amounts between different currencies using up-to-date exchange rates. It's ideal for - travelers, traders, or anyone who needs quick and accurate currency conversions.

# Features

- Convert between over 150 international currencies
- Real-time exchange rate updates via external API
- User-friendly and responsive interface

# Requirements

- Internet connection (for fetching exchange rates)

# Installation

- Clone the repository:
- Shellgit clone https://github.com/your-username/Currency-Converter-App.git

# Usage

- Select the source and target currencies.
- Enter the amount you want to convert.
- Click the “Convert” button to see the result.

# API Used

- This app uses ExchangeRate API (or another relevant API) to fetch the latest exchange rates.
